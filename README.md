# E-Learning Platform - Java OOP Project

## System Definition

### Object Types
- User
- Student
- Instructor
- Course
- Lesson
- Quiz
- Enrollment
- Task
- StudyTask
- Progress

## Supported Actions
1. Add user
2. Remove user
3. Add course
4. Remove course
5. Find course
6. Enroll student
7. Add lesson
8. Remove lesson
9. Create quiz
10. Solve quiz
11. Calculate quiz score
12. Add task
13. Remove task
14. View courses
15. View student progress
16. Sort courses
17. View categories
